# Builder V2 parity project — completed

The production-parity project was accepted and released on 2026-07-21.

The completed work included the builder layout and authoring tools, saved
lessons, exports, handouts, presenter toolbar and interactions, PDF rendering,
polling and confidence summaries, camera persistence, starter quadrant
controls, administration, tablet behaviour, and student sharing. Manual
acceptance testing and automated unit, contract, visual, presenter-runtime,
and deployment smoke coverage passed before cutover.

The unified application now lives at `/builder`. The `/builder-v2` URL is a
compatibility redirect, not a second implementation. Historical implementation
details remain available in Git history; current operational instructions are
in `docs/operations/production-release.md`.
